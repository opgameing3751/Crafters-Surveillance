# Crafters-Surveillance
-------------------------
This is a CLI program that allows users to acess player tab outside of the game of Minecraft


# How to use
--------------
to use Crafters-Surveillance you first have to open that Settings.env file and put in your api key.
you can get and api key by following the this page https://wiki.crafters.one/wiki/API_Usage under **Obtaining an API Key**
after that you just need to run the start sh or running Main.py

all dependencies should auto install
